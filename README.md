# CodeCamp 2016

## Dockerfile

```sh
docker run \
	-it \
	-d \
	-v /path/to/your/app:/data \
	--name codecamp \
	f3l1x/codecamp
```

Great and simple docker container based on Debian Jessie with pre-configured services.

**Services**

- nodejs 5.x
- npm
- ionic
- cordova
- gulp
